Credit Card Fraud Detection Using ANN

Project Overview

This project focuses on detecting fraudulent credit card transactions using Machine Learning. An Artificial Neural Network (ANN) model is used to analyze transaction patterns and predict whether a transaction is genuine or fraudulent.

Objective

The main objective of this project is to build a system that can identify fraud transactions and help in reducing financial losses in digital payment systems.

Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- TensorFlow / Keras
- Streamlit

Dataset

The dataset contains credit card transactions with features such as Time, Amount, and anonymized variables (V1–V28). These features help the model learn transaction patterns.

Model Used

Artificial Neural Network (ANN) is used to train the model for fraud detection.

Application

A Streamlit web application is created where the user can enter transaction details like time and amount, and the system predicts whether the transaction is fraud or legitimate.

How to Run the Project

1. Install required libraries
   pip install -r requirements.txt

2. Run the Streamlit app
   streamlit run app.py

3. Open the local link shown in the terminal.

Output

The system predicts whether the transaction is:

- Legitimate Transaction
- Fraudulent Transaction
